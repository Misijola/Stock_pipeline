# Stock Market Pipeline

A data engineering project that extracts live stock data from Alpha Vantage API, 
transforms it using Python and pandas, and loads it into PostgreSQL.

## Tech Stack
- Python
- PostgreSQL
- Pandas
- Grafana

## How to Run
1. Clone this repo
2. Add your API key to .env
3. Run fetch_stocks.py → transform_stocks.py → load_stocks.py
